WEB101 ADVOCACY PROJECT


Project Summary

Program/Skills Used: Replit, HTML, CSS, Javscript

Made during my time in the Codepath WEB 101 online course in Spring 2023. The website's main focus is to raise awareness about the impacts of climate change. The course taught me the fundamentals of front-end design and the process involved in creating a website, including HTML, CSS, and Javascript. The website I created has a fully functional petition, a navigation bar, and a dark mode for easy viewing.

Lessons Learned

I had primarily been coding with Python before taking this course, so switching to Javascript was quite intimidating at first. I faced difficulty in defining functions and encountered other syntax-related problems. Fortunately, the tech fellows who were part of the course's help team were able to assist me in clearing up any confusion I had. My biggest challenge was getting my petition to function correctly on my website. I encountered problems with the pop-up text that appears after someone signs the petition. Instead of displaying the actual text I wanted to show up, only the placeholder text was visible. Luckily, a tech fellow helped me correct my function's code and we were finally able to get the event listener to work properly. Overcoming these kinds of challenges helped me gain a better understanding of Javascript by the end of the semester.

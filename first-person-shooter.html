3D FIRST-PERSON SHOOTER

Project Summary

Program/Skills Used: Unity 3D, Visual Studio, C#

As part of my Game Development I class at GSU, I created a 3D shooter game. The game features two types of weapons - a gun and a missile launcher - which can be switched by pressing the "Q" key. The game also includes enemy NPCs that are controlled by AI and can chase the player, a health/bullet system, and a win/lose scenario. The game was developed using Unity 3D with assets from the Unity Store, and the scripts were coded in C#.

Lessons Learned

As a beginner in C# coding and game development in Unity 3D, there are a few issues in my game that need fixing. For instance, collectible bullets cannot be added to the inventory, and the number of bullets goes into negative after the player runs out of bullets. However, this experience has taught me several new skills related to game development. I have learned to code in C#, which forms the basis of most functionalities in the games. Moreover, I have learned the proper practices of saving assets in Unity 3D. Unfortunately, I lost my progress in the final project while importing assets, but I have learned from my mistake.
